# OIBSIP Internship Projects

This repository contains the Python projects completed as part of the OIBSIP internship.

---

## 📌 Task 1: BMI Calculator
- Developed using Python and Tkinter
- Calculates Body Mass Index (BMI)
- Displays category:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Includes input validation and error handling

---

## 📌 Task 2: Password Generator
- Generates strong random passwords
- User can select:
  - Letters
  - Numbers
  - Symbols
- Includes copy-to-clipboard feature
- Built with Tkinter GUI

---

## 📌 Task 3: Weather App
- Fetches real-time weather data using OpenWeather API
- Displays:
  - Temperature
  - Humidity
  - Weather condition
- Built using Python, Tkinter, and Requests

---

## 🛠 Technologies Used
- Python
- Tkinter (GUI)
- Requests (API handling)
